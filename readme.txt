== Description ==

Unfortunately, this plugin is not supported anymore. We recommend you use the [Suggestion Toolkit](https://wordpress.org/plugins/suggestion-toolkit/) plugin. It provides extended functionality.

== Installation ==

Install [Suggestion Toolkit](https://wordpress.org/plugins/suggestion-toolkit/) plugin instead.
